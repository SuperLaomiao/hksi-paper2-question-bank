# HKSI Paper 2 Question Bank

Mobile-first HKSI LE Paper 2 practice question bank for Securities Regulation revision.

## What is included

- 800 original practice questions for HKSI Paper 2
- Chapter filters for all 7 Paper 2 syllabus areas
- Practice mode, review mode, mock exam mode, bookmarks, wrong-question review, and answer sheet
- Official/public reference links for HKSI, SFC, HKEX, and Hong Kong e-Legislation materials

## Important notice

This is an original revision tool based on the HKSI Paper 2 syllabus and public regulatory materials. It is not an official HKSI product and does not reproduce official proprietary question banks.

## Local use

```bash
npm start
```

Then open:

```text
http://localhost:3000/public/hksi-paper2.html
```
